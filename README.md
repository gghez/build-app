# build-app

## Lettre de mission

**Décrire des processus de livraison IT visuellement.**

L'objectif de ce projet est de fournir une application web permettant de modéliser, documenter et visualiser des processus de livraison IT (pipelines CI/CD, déploiements, workflows de release, etc.) de manière graphique et intuitive.

### Stack technique

- **Frontend / Backend** : Application web Node.js
- **Base de données** : PostgreSQL

### Principes

- **Visuel avant tout** : chaque processus de livraison doit être représenté sous forme de diagramme ou de graphe interactif.
- **Traçabilité** : chaque étape d'un pipeline doit être documentée et reliée aux outils, environnements et acteurs concernés.
- **Ouvert** : les processus décrits doivent pouvoir être exportés et partagés facilement.
